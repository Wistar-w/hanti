# 🌱 孩子瞎打哒小测试 — 写给朋友的人格探测仪

> 一个温柔、不严肃、带点个人色彩的 MBTI 风格小测试。  
> 每道题都像聊天，最后会给你一个可爱的类型名字和一张配图。

## 📦 这是什么？

这是我为一个朋友写的网页小工具。  
它基于 MBTI 的四维度模型（I/E, N/S, T/F, J/P），但**刻意做得不学术、不评判**。  
12 道题，一题一页，选完自动出结果，最后还有配图和一句悄悄话。

## ✨ 特点

- 🧸 **温柔提问** — 没有“你必须”“你总是”，只有“你可能”“你更倾向于”。
- 🎴 **一题一页** — 不用一次面对所有题目，适合容易焦虑的人类。
- 🖼️ **结果配图** — 16 种结果对应 16 张本地图片（你可以换成自己的画或照片）。
- 🌄 **全局背景** — 支持自定义背景图，让测试氛围更安静。
- 🔀 **题目乱序** — 每次开始测试或重测，题目顺序都会随机打乱，避免肌肉记忆。

## 🛠️ 如何修改为自己用的版本？

1. **改背景图**  
   打开 `index.html`，搜索 `background-image: url('images/bg.jpg')`，换成你的图片路径。

2. **改结果图片**  
   在 `typeDetails` 对象里，把每个类型的 `img` 路径改成你实际存放的图片文件名（支持 `.png`, `.jpg` 等）。

3. **改题目文字**  
   在 `originalQuestions` 数组里，修改 `text` 和 `options` 的内容。可以写得更像你说话的语气。

4. **改结果描述**  
   在 `typeDetails` 里修改 `desc` 字段，给朋友写一些只有你们才懂的悄悄话。

## 🌍 如何发布到网上让别人也能测？

- **用 GitHub Pages**：把整个文件夹推送到 GitHub 仓库 → Settings → Pages → 选择 main 分支 → 保存 → 几分钟后获得网址。
- **用 Netlify**：直接把文件夹拖拽到 [Netlify Drop](https://app.netlify.com/drop) 即可。
